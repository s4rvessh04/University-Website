Get Started

Welcome developer
In order to use this api you need to register to our developers program ->

Basics

The bare minimum to get you started.

  • Install "requests library", run pip install requests on console.
  • Create a file named api.py
  • Make imports.
    import request
    import json
    from pprint import pprint
  • Make attributes for api as
    headers = {}
    params = {}

Tokens

Authentication requirement.

  • Paste your token in headers
  • It should look like headers = {'Authorization': 'Token your-token-here'}

NOTE: There must be a whitespace between Token and your-token

Querying

Getting data from the api.

For making a query to api, pass the query field along with the query in params
Example: params = {'query_field':'query'}

Query Fields

The only link on which you can query data is academic-list.
To query certain student, subject or semester:

  • For student: 'student__name'
  • For subject: 'subject__name'
  • For semester: 'semester__name'

NOTE: The query must be case sensitive.

Example: If you want to query a student then params = {'student__name': 'FirstName MiddleName LastName'} With the whitespace.

Links

User accessable links.

To get all accessible links. Use this link {{ link_link }}. It is suggested to use a dictionary to hardcode this link in your python file.
Like API_URLS = {'links': '{{link_link}}'}

academic-list

It will return all students academic data.

Fields Include
  • id
  • student
  • semester
  • subject
  • marks

NOTE: This object will also contain your data. But if you want to see your academic record use the next field.


my-academics

It will return academic data related to you.

Fields are the same as academic-list.


my-student-profile

It will return your student profile data.

Fields Include
  • id
  • branch
  • name
  • phone

account-details

It will return all the account related details.

Fields Include
  • id
  • username
  • date_joined
  • email
  • api_token
